# 产品背景与体验目标

## 目标用户
- 习惯于用一个统一界面管理生活多个方面（娱乐、目标、学习、日程、日记）的个人。
- 期望通过 AI 助手获得灵感、建议，减少在不同应用之间切换。

## 核心体验
1. **统一工作台**：左侧栏 + 移动端导航切换不同功能页面，整体风格温暖、卡片化、多留白。
2. **AI 助手**：浮动按钮 + 侧边抽屉/全屏面板，支持上下文对话、可拖拽调整宽度。
3. **模块化场景**：娱乐推荐、目标管理、学习计划、日程安排、日记等页面均围绕“个人效率/灵感”展开。
4. **设置中心**：启用 AI 面板、选择主题、通知、知识库来源，模拟模型配置和连接状态。

## 用户痛点与解决方案
- **信息分散** → 通过 App Router 各页面 + 共享布局聚合。
- **灵感/建议缺乏** → AI 助手及娱乐推荐提供建议与收藏。
- **跨设备体验** → 移动端底部导航 + 响应式布局适配。

## 成功标准
- 用户能够在不同模块间无缝切换，交互顺畅。
- AI 功能启用后随处可见，打开/关闭状态可控。
- 各模块提供示例数据，便于演示功能与视觉风格。
