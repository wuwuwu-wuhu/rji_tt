# 项目概览

- **名称**：LifeLog AI
- **类型**：全栈智能生活日志管理应用
- **目标用户**：希望借助AI助手统一管理生活多个方面（日记、娱乐、学习、日程等）的个人用户
- **核心价值**：通过统一的界面与AI辅助能力，让用户高效管理日常生活、学习目标、日程计划及娱乐推荐

## 项目愿景

LifeLog AI旨在成为用户的智能生活助手，通过AI技术整合日常生活的各个方面，提供个性化的建议和高效的管理工具，帮助用户实现更好的生活平衡和个人成长。

## 核心功能

1. **AI助手面板**
   - 可拖拽浮动按钮
   - 可调整宽度的侧边面板
   - 支持全屏模式
   - 多轮对话支持
   - 会话历史管理

2. **日记管理**
   - 时间线展示
   - 按日期分组
   - 标题和内容编辑
   - 心情和标签支持
   - 私密设置

3. **娱乐推荐**
   - 多类型推荐（电影、书籍、活动、游戏）
   - 收藏功能
   - 自定义添加
   - 分类浏览
   - 评分系统

4. **目标管理**
   - 长期/中期/短期目标分类
   - 进度跟踪
   - 里程碑设置
   - 目标日志记录

5. **日程管理**
   - 周视图展示
   - 时间段管理
   - 事件分类
   - 提醒功能

6. **个性化设置**
   - AI助手开关
   - 模型配置管理
   - 主题设置
   - 通知设置

## 技术架构

### 前端技术栈
- **框架**: Next.js 16 + React 19 (App Router)
- **样式**: Tailwind CSS 4 + 自定义UI组件库
- **UI组件**: Radix UI生态
- **状态管理**: React Context + 本地状态
- **类型安全**: TypeScript
- **图标**: Lucide React
- **包管理**: pnpm

### 后端技术栈
- **框架**: FastAPI (Python)
- **数据库**: PostgreSQL + SQLAlchemy ORM
- **缓存**: Redis
- **认证**: JWT令牌认证
- **API文档**: Swagger UI + ReDoc
- **外部服务**: OpenAI API集成
- **任务队列**: Celery (配置中)
